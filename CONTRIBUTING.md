# Contributing to DDS

Start with the [DDS community contributing guide](https://github.com/dds-wg/.github/blob/main/CONTRIBUTING.md). It is the shared guide for issues, pull requests, conduct, governance, and maintainer expectations.

Join the [W3C DDS Community Group](https://www.w3.org/community/dds/) to stay informed about specification work, calls, and Community Group process. Specification contributions follow the W3C DDS Community Group process and are governed by the [W3C Community Contributor License Agreement](https://www.w3.org/community/about/process/cla/).

This repository has separate terms for specifications, website code, and proof-of-concept software. See [COPYING-README.md](./COPYING-README.md) for the repository licence map.
